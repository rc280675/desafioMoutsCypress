# Automação de Testes - Serverest

Projeto de automação utilizando Cypress para validar frontend e API da aplicação Serverest.

## Tecnologias
- Cypress
- JavaScript
- Page Object Model

## Cenários automatizados

### Frontend
- Fluxo de Compra do ambiente SauceDemo

### API
- Fluxo de reserva no Booking com restFul

## Execução

Instalar dependências

npm install

Rodar testes

npx cypress open > para modo gráfico ou
npx cypress run > modo headless
para executar com report > npm run test:report
